# States with a select form
Import the component in Formik or html form

```
import StatesUsa from './StatesUsa'
```

**Formik**
``` 
<Field
  component="select"
  id="state"
  name="age"
  placeholder="Your State"
>
  <StatesUsa />
</Field>
```
**HTML**
```**
<select>
  <StatesUsa />
</select>
```
